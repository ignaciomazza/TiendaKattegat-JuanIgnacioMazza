Buenas, soy Juan Ignacio Mazza, el desarrollador detras del proyecto Tienda Kattegat.

El proyecto realizado es un ecommerce interactivo desarrollado con la tecnologia de React Js, la idea es una simulacion de una tienda en linea de cervezas.
Es completamente adaptable a smartphones, tablets, notebooks y computadoras de escritorio.
Se pueden realizar compras, las cuales seran guardadas en una base de datos.
